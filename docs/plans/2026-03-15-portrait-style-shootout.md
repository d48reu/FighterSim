# Fighter Portrait Style Shootout

Use one fictional target across all styles so the comparison is about art direction, not subject drift.

Shared subject:
- male MMA fighter
- age 28
- lightweight striker
- shoulders-up headshot
- intense calm expression
- dark premium sports-broadcast environment
- no text, no logos, no props

## Style A — Premium Illustrated Broadcast
Prompt:
"Stylized premium sports-broadcast illustrated portrait of a professional MMA fighter, male lightweight striker age 28, shoulders-up headshot, intense calm expression, athletic build, sharp facial structure, dark studio background with subtle cyan broadcast accent lighting, posterized realism, premium sports game profile art, controlled dramatic rim light, clean silhouette, centered composition, no text, no logos, no gloves, no belt, no props, high consistency"

Why test it:
- closest to sports-game profile art
- safest coherent direction with current UI

## Style B — Semi-Real Stylized Studio
Prompt:
"High-end semi-real stylized portrait of a professional MMA fighter, male lightweight striker age 28, shoulders-up headshot, neutral intense expression, athletic face, premium dark studio backdrop, subtle cinematic lighting, realistic anatomy but slightly art-directed finish, clean skin treatment, premium fight-promotion media-day portrait, centered composition, no text, no logos, no props, no uncanny hyperdetail"

Why test it:
- tests whether the user wants more realism without full fake-photo uncannyness

## Style C — Gritty Fight-Poster Graphic
Prompt:
"Gritty premium fight-poster portrait of a professional MMA fighter, male lightweight striker age 28, shoulders-up headshot, stern expression, athletic presence, dark textured arena backdrop, bold contrast, subtle cyan and red promotional lighting, graphic posterized realism, premium combat sports key art, dramatic but controlled, centered composition, no text, no logos, no gloves, no belt, no props"

Why test it:
- more attitude and event-poster energy
- less profile-card clean, more main-event heat

## Review Criteria
- Does it feel premium?
- Does it fit the Arena UI?
- Does it look like a game asset instead of random AI art?
- Does it avoid uncanny face problems?
- Which style would still hold up across 100+ fighters?
